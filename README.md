## üöÄ How to Run the Project

### üñ•Ô∏è Frontend (Angular 19)

1. Open terminal and navigate to the frontend folder:

    ```bash
    cd frontend
    ```

2. Install all the node modules:

    ```bash
    npm install
    ```

3. Run the Angular project:

    ```bash
    ng serve --open
    ```

4. By default, the app will be running at:

    ```
    http://localhost:4200
    ```

---

### ‚öôÔ∏è Backend (Spring Boot - Eclipse)

1. Open Eclipse IDE.

2. Import the backend project:
    - `File` ‚Üí `Import` ‚Üí `Existing Maven Project` ‚Üí Select the `backend` folder.

3. Configure your **application.properties** with PostgreSQL settings:

    ```properties
    spring.datasource.url=jdbc:postgresql://localhost:5432/your_database
    spring.datasource.username=your_username
    spring.datasource.password=your_password
    spring.jpa.hibernate.ddl-auto=update
    ```

4. Run the Spring Boot application:
    - Right-click on the project ‚Üí `Run As` ‚Üí `Spring Boot App`.

5. Backend will start on:

    ```
    http://localhost:8080
    ```
